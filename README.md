# Infobase documentation (Mintlify)

OneInbox Infobase guide — content from `infobase.mdx`.

## Screenshots

Add these files under `images/` for screenshots to render:

- `infobase-01-overview.png` through `infobase-17-agent-saving.png` (see `index.mdx` for full list)

## Local preview

```bash
npm i -g mint
cd infobase-docs
mint dev
```

## Publish

Connect this folder to [Mintlify](https://mintlify.com/start) via GitHub and push to deploy.
