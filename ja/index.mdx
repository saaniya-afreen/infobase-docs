---

## 概要

**Infobase** は OneInbox のナレッジ管理レイヤーです。ウェブページやアップロードしたドキュメントから得た構造化情報を AI 音声エージェントに直接紐づけられます。紐づけ後、エージェントは通話中にリアルタイムでこのコンテンツを参照し、質問への回答、反論への対応、文脈に沿った正確な会話が可能になります。

<Note>
  Infobase は **フォルダ** 単位で整理されます。各フォルダには、スクレイピングした URL とアップロードしたファイルを混在させることができ、フォルダ全体を 1 つ以上のエージェントに添付できます。
</Note>

---

## ステップ 1: Infobase を開く

左サイドバー下部の **Infobase アイコン**（積み重ねたレイヤーアイコン）をクリックします。Infobase 画面が開き、すべてのフォルダとその中のファイルが表示されます。

<img
  src="/images/Screenshot_2026-05-26_at_12_39_43.png"
  alt="Business フォルダに処理中のファイルが 1 件ある Infobase の概要画面"
/>

> この例では、**Business** フォルダに `Webinar_AI SDR Workshop.pptx` が 1 件あり、ステータスは **In Progress**（処理中）です。

---

## ステップ 2: フォルダを追加する

ウェブサイトやファイルを追加する前に、ナレッジベースを整理するフォルダを作成します。

1. 左側の **Infobases** メニューで、**Infobases** 見出しの横にある **+** ボタンをクリックします。
2. フォルダ名を入力します（例: `Business` や `Product Docs`）。
3. 確定してフォルダを作成します。

新しいフォルダが左の Infobases 一覧に表示されます。フォルダをクリックしてメイン画面で開き、次のステップでコンテンツを追加します。

<img
  src="/images/Screenshot_add_folder_menu.png"
  alt="Infobases 見出し横のプラスボタンで新しいフォルダを作成する画面"
/>

---

## ステップ 3: フォルダに情報を追加する

作成したフォルダを開き、フォルダ画面の右上にある **+ Add Info** をクリックします。

モーダルに 2 つのオプションが表示されます。

<img
  src="/images/Screenshot_2026-05-26_at_12_39_53.png"
  alt="Website と Upload File のオプションがある Add Info モーダル"
/>

| オプション | 説明 |
|---|---|
| **Website** | URL を指定すると、OneInbox がサイトをクロールしてページを自動でインデックスします |
| **Upload File** | PDF、DOCX、TXT、CSV 形式のドキュメントをアップロードします |

該当するオプションを選び、**Next** をクリックします。

---

## ステップ 4a: ウェブサイトを追加する

**Website** を選択すると、URL の入力画面が表示されます。

<img
  src="/images/Screenshot_2026-05-26_at_12_40_12.png"
  alt="https://oneinbox.ai/ が入力された URL 入力欄"
/>

ルートドメインまたは特定ページの URL を入力し、**Next** をクリックします。OneInbox がサイトをクロールし、検出されたすべてのページの一覧を返します。

### インデックスするページを選択する

クロール後、検出された URL のチェックリストが表示されます。初期状態ではすべてのページが選択されています。

<img
  src="/images/Screenshot_2026-05-26_at_12_40_24.png"
  alt="300 ページ中 59 ページがチェックされた URL 選択リスト"
/>

- **Select all** で全ページを含めるか、個別にチェックをオン/オフできます。
- **Filter URLs** で一覧内の特定ページを検索できます。
- カウンター（例: `59/300`）は、上限内で選択されているページ数を示します。

**Next** をクリックして選択を確定します。

<img
  src="/images/Screenshot_2026-05-26_at_12_40_34.png"
  alt="緑のチェックマーク付きで確定した URL 一覧と追加ソースのオプション"
/>

<Tip>
  この画面から、追加のウェブサイトやファイルを続けて登録してから **Next** で確定できます。
</Tip>

---

## ステップ 4b: ファイルをアップロードする

**Upload File** を選択したら、**Upload File** カードをクリックすると、OS のファイル選択ダイアログが開きます。

<img
  src="/images/Screenshot_2026-05-26_at_12_44_26.png"
  alt="Upload File オプションがハイライトされた Add Info モーダル"
/>

<img
  src="/images/Screenshot_2026-05-26_at_12_44_50.png"
  alt="Oneinbox.pdf が選択された macOS のファイルピッカー"
/>

アップロードするファイル（PDF、DOCX、TXT、CSV）を選び、**Open** をクリックします。

モーダル内にファイルが表示され、緑のチェックマークでアップロード待ち状態であることが確認できます。

<img
  src="/images/Screenshot_2026-05-26_at_12_44_57.png"
  alt="Oneinbox.pdf がチェックマーク付きでステージングされた Add Info モーダル"
/>

**Next** をクリックして確定し、処理を開始します。

---

## ステップ 5: 処理ステータスを確認する

ソースを送信すると、フォルダ画面に戻ります。追加したファイルと URL は次のいずれかのステータスで一覧に表示されます。

<img
  src="/images/Screenshot_2026-05-26_at_12_44_13.png"
  alt="60 件すべてが In Progress の Infobase フォルダ"
/>

| ステータス | 意味 |
|---|---|
| **In Progress** | コンテンツをクロールまたは処理中 |
| **Ready** | インデックス完了。エージェントが利用可能 |

<img
  src="/images/Screenshot_2026-05-26_at_12_46_18.png"
  alt="Oneinbox.pdf と URL が Ready になった Infobase フォルダ"
/>

### ステータスの更新

ページを再読み込みせずにステータスを確認するには、フォルダ上部のステータスバッジ横の **更新アイコン**（↻）をクリックします。

<img
  src="/images/Screenshot_2026-05-26_at_12_46_49.png"
  alt="更新アイコンにホバーしたときの Refresh status ツールチップ"
/>

## ステップ 6: フォルダにユーザーを割り当てる（任意）

Infobase フォルダにチームメンバーを割り当てられます。更新ボタン横の **+** アイコンをクリックして、ユーザー割り当てドロップダウンを開きます。

<img
  src="/images/Screenshot_2026-05-26_at_12_49_18.png"
  alt="プラスアイコンにホバーしたときの Assign user ツールチップ"
/>

<img
  src="/images/Screenshot_2026-05-26_at_12_49_36.png"
  alt="チームメンバーとエージェントの一覧があるユーザー割り当てドロップダウン"
/>

ドロップダウンから 1 人以上を選択します。アクセス制御やナレッジベースの担当整理に便利です。

---

## ステップ 7: エージェントに Infobase を添付する

Infobase フォルダの準備ができたら、任意のエージェントに添付できます。

1. 左サイドバーから **Agents** を開きます。
2. 設定するエージェントを選択します。
3. **Role** タブを開きます。
4. 右側の **Add attachment** パネルを確認します。
5. **Select file** をクリックし、ドロップダウンから Infobase フォルダを選択します。

<img
  src="/images/Screenshot_2026-05-26_at_12_51_11.png"
  alt="Business の Infobase が選択可能な Add Attachment パネル（Role タブ）"
/>

フォルダが添付一覧に表示されます。

<img
  src="/images/Screenshot_2026-05-26_at_12_51_18.png"
  alt="Business の Infobase が添付された Role タブ"
/>

6. **Save** をクリックして変更を保存します。

<img
  src="/images/Screenshot_2026-05-26_at_12_51_29.png"
  alt="Business の Infobase 添付後、Save ボタンがローディング状態のエージェント画面"
/>

<Check>
  エージェントはライブ通話中に Infobase のコンテンツを参照できるようになります。
</Check>

---

## ステップ 8: ファイルを削除する

フォルダ内の特定のファイルまたは URL を削除する手順です。

1. 対象のフォルダを開き、一覧から削除したい行を見つけます。
2. 行の右端にある **ゴミ箱アイコン** をクリックします。
3. 確認ダイアログで削除を確定します。

<img
  src="/images/Screenshot_delete_file.png"
  alt="行の右端に削除用ゴミ箱アイコンがある Infobase のファイル一覧"
/>

<Warning>
  ファイルを削除すると、フォルダから完全に削除されます。エージェントに添付済みの場合、通話時にそのコンテンツは参照されなくなります。
</Warning>

---

## ステップ 9: フォルダを削除する

Infobase フォルダ全体（中のファイルすべて）を削除する手順です。

1. 削除するフォルダを開きます（または左の **Infobases** 一覧から選択します）。
2. フォルダ画面右上の **+ Add Info** の横にある **⋮**（三点メニュー）をクリックします。
3. ドロップダウンから **Delete** を選択します。
4. 確認ダイアログで削除を確定します。

<img
  src="/images/Screenshot_delete_folder.png"
  alt="Delete オプションが表示されたフォルダのメニュー"
/>

<Warning>
  フォルダを削除すると、含まれるすべてのファイルと URL が削除されます。添付の不整合を避けるため、必要に応じて事前にエージェントからフォルダの添付を解除してください。
</Warning>

## まとめ

| ステップ | 操作 |
|---|---|
| 1 | サイドバーから Infobase を開く |
| 2 | 左メニューの **Infobases** 横の **+** でフォルダ名を入力し作成 |
| 3 | フォルダを開き **+ Add Info** をクリック |
| 4 | ウェブサイト URL を追加するかファイルをアップロード |
| 5 | ステータスが **In Progress** から **Ready** になるまで待つ |
| 6 | （任意）フォルダにチームメンバーを割り当て |
| 7 | Role タブからフォルダをエージェントに添付して保存 |
| 8 | 各行のゴミ箱アイコンで個別ファイルを削除 |
| 9 | **⋮** メニュー → **Delete** でフォルダ全体を削除 |

<Warning>
  **In Progress** のファイルはまだインデックスされていません。すべてのソースが **Ready** になる前にエージェントへ添付すると、通話時のナレッジが不足する場合があります。
</Warning>
